# 內建副本說明

此資料夾為 **CourseFlow 專案內建** 的 `web-video-presentation` Skill，與上游 [garden-skills](https://github.com/ConardLi/garden-skills) 同步維護。

- **用途**：`npm run courseflow -- wvp scaffold`、Studio「建立 presentation 腳手架」
- **更新**：在 repo 根目錄執行 `node scripts/sync-wvp-skill.mjs`
- **不必設定** `WVP_SKILL_DIR`（除非要強制覆蓋為其他路徑）
