# 如何使用 web-video-v2 Skill

## 這是什麼？

`web-video-v2` 是一個 Claude Agent Skill，能把一篇文章或口播稿，變成**點選驅動的 16:9 網頁演示**——看起來像影片，但實際上是可互動的網頁。適合錄屏教學、產品 demo、B 站 / YouTube 影片、或有電影感的 talk。

## 快速開始

### 1. 準備你的素材

只需要一篇文章或口播稿。例如：

```
我想做一個介紹 React Hooks 的教學影片，大約 5 分鐘，
目標觀眾是前端初學者。
```

### 2. 啟動 Skill

在對話中告訴 Claude：

> 幫我把這篇文章做成影片

Claude 會自動調用 `web-video-v2` skill，進入以下流程：

### 3. 四階段工作流

| 階段 | 做什麼 | 你需要做什麼 |
|------|--------|-------------|
| **Phase 1** 內容編寫 | Claude 產出口播稿 `script.md` + 開發計劃 `outline.md` | 確認稿子、主題、素材、開發模式 |
| **Phase 2** 網頁開發 | 逐章生成 Vite + React + TS 網頁 | 驗收第 1 章，後續章節自動推進 |
| **Phase 3** 音訊合成 | 可選：用 IndexTTS2 或 MiniMax 合成口播 | 決定是否需要音訊 |
| **Phase 4** 錄屏後期 | 用瀏覽器全螢幕錄屏 | 錄屏、剪輯 |

### 4. 關鍵的「一次對齊」

Phase 1 結束後，Claude 會停下來請你確認 5 件事：

1. **稿子** — 口播稿節奏對不對？
2. **Outline** — 章節切分合理嗎？
3. **主題** — 想要什麼視覺風格？
4. **素材** — 有沒有 logo、截圖、程式碼要放進去？
5. **開發模式** — 逐章確認 / 順序開發 / 並行開發？

## 內建 6 大能力

產出的網頁自帶這些功能，不需要額外設定：

- 🎬 **底部字幕條** — 顯示當前步驟的口播文字
- 🎛️ **頂部功能表** — 暫停 / 播放 / 自動播放 / 速度切換 / PDF 下載
- 📄 **一鍵匯出 PDF** — 每步一頁，可當簡報用
- 🔢 **右下角頁碼** — `2.3` = 第 2 章第 3 步，方便對話定址
- 🖥️ **可選全螢幕** — 加 `?fit=cover` 填滿畫面無黑邊
- ✅ **自動排版自檢** — Playwright 檢查字級和溢出

## 實用 URL 參數

```
?auto=1         # 自動播放全部
?fit=cover      # 全螢幕無黑邊
?recording=1    # 錄屏模式（隱藏控制元件）
?subs=off       # 隱藏字幕條
```

## 對話定址技巧

開發過程中可以用頁碼精準修改：

- 「重寫 2.3」→ 重寫第 2 章第 3 步的畫面
- 「2.1 的字太小」→ 調整特定步驟的排版
- 「第 3 章整體太快」→ 調整整章節奏

## 適合什麼場景？

- ✅ 把技術文章變成教學影片
- ✅ 產品功能演示
- ✅ Conference Talk / Keynote
- ✅ 課程內容視覺化
- ✅ 任何「想要影片感但不想開 PR / AE」的場景
