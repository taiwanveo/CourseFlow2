## 建議流程

| 步驟 | 頁面 | 要測的功能 |
|------|------|------------|
| 1 | 內容 | 只貼**學員文稿** → AI 生成大綱（約 6～7 章） |
| 2 | 大綱 | Hook 章上傳 **3 張開場圖**（Checkpoint 素材或步驟配圖 2–4） |
| 3 | 配音 | **批次配音**；切到其他分頁 → 瀏覽器通知 |
| 4 | Craft | **批次 Craft**；第四章逐步看 chart／table 建議 |
| 5 | Craft | 第五章前一步產 **解說動畫** → 看「步間轉場建議無」 |
| 6 | Craft | 章間轉場：Hook→flow 應見 **wipe → slide-right** 建議 |
| 7 | 發佈 | **WVP 打包**；失敗時看章節／步驟／API 原文 |
| 8 | 發佈 | **一鍵分享連結** → 無痕開 `/watch/{slug}?auto=1` |
| 9 | 維運 | `GET /api/health` 核對 `buildSha`、`workerOnline` |

## 學員文稿 → 系統預期（對照表）

| 學員文稿章節 | 預期版型 | 要驗的功能 |
|--------------|----------|------------|
| 開場 / 第一章 | `hook` | A1 開場圖、章間 **wipe**、短句 **drop-in**（≤28 字） |
| 第二章 製作流程 | `flow` | 流程關鍵字、`slide-right` 章間建議 |
| 第三章 三個原則 | `list-reveal` | A2 清單拆步（≥10 字）、章間 **fade** |
| 第四章 數據視覺 | dataVisual / visual-mix | 見下表「圖表觸發對照」 |
| 第五章 概念與數據 | chart + 解說動畫 | 並存、步間 **none** |
| 第六章 / 結語 | 一般章節 | 分享連結、免登入 `/watch` |

### 第四章 — 圖表觸發對照（啟發式，勿改數字標點）

| 小節 | 口播關鍵句 | 預期 VisualConfig |
|------|------------|-------------------|
| 方案怎麼選 | `方案對照：方案A：成本 12…` | **table**，`sortBy.key` = 成本 |
| 費用怎麼分 | `研發 40%、行銷 25%…` | **pie** |
| 一個關鍵數字 | `轉換率達到 38%` | **kpi** |
| 成長是不是穩定 | `1月100…6月420` | **line** |
| 季度表現 | `Q1 120億…Q4 180億` | **bar** 或季度序列 |
| 為什麼要同時看… | 長句（≥48 字） | 進場 **rise-soft** |

> AI 切步後若第四章步數不足 6 步，請在大綱頁手動拆步，每小節一步。

### 第五章 — 解說動畫（手動步驟）

1. 在 Craft「步驟配圖」為**第五章第一步**（或「例如：當我們談裝置…」那步）產生解說動畫。  
2. 同一步口播含 `占比：桌面 55%…` → 打包後應 **chart 與動畫並存**。  
3. UI 應提示該步**步間轉場建議設為無**。

## 可選：故意製造失敗（長任務 UX）

- API Key 無效或斷網時重跑批次配音／打包  
- 確認：Toast／面板有**章名＋步驟＋API 錯誤原文**；發佈頁有 `lastError`；背景分頁有瀏覽器通知
